# PartialSum
## 動的計画法（DP)
部分和問題の解法

n個のせいの整数a[0], a[1], …, a[n-1]と正の整数Aが与えられる。<br>
これらの整数を選んで総和がAになるようにすることが可能か判定せよ。<br>
可能ならば”Yes”と出力し、不可能ならば”No”と出力せよ。
<br>
【制約】<br> 
1≤n≤100 <br>
1≤a[i]≤1000<br> 
1≤A≤10000 <br>

【数値例】 

1) 
n = 3 <br>
a = (7, 5, 3) <br>
A = 10 <br>

	答え　Yes 
 
2) 
n = 2 <br>
a = (9, 7) <br>
A =6 <br>
 
	答え　No
